# Movies Lists

A responsive Vue.js application to view the list of recent movies and to get a random movie to watch!

Features
- [x] List of movies with ordering;
- [x] Movies details;
- [x] Show similar movies;
- [x] Get a random movie!;
- [ ] Search movie;
- [ ] Movie recommendations system
